# cnssl-leyu-lab

本仓库用于归档和发布多个独立的 HTML 页面。  
这些页面可能包含示例、工具页面、实验性内容或静态展示页，不针对任何特定网站或域名提供服务。

## 项目简介

`cnssl-leyu-lab` 是一个轻量级页面归档仓库，主要用于存放和展示一些独立的 HTML 文件。  
仓库内容不涉及商业用途，仅供学习、参考或个人项目归档使用。

## 目录说明

```
.
├── index.html          # 仓库根页面（可选）
├── pages/              # 归档页面存放目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── README.md           # 本文件
```

- `pages/` 目录下存放各类独立 HTML 页面。
- 每个页面均为独立文件，不依赖其他资源（除非页面内自引用）。
- 根目录下的 `index.html` 可用于简单导航或跳转说明。

## 页面归档说明

- 所有页面均为静态 HTML，可直接在浏览器中打开。
- 页面内容可能包含实验性代码、临时工具、笔记或演示页面。
- 仓库不保证所有页面长期可用或内容更新。
- 页面命名尽量清晰，但文件名不代表最终内容。
